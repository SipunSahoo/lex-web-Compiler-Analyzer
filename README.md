# Lex Web Tool

A web-based Lex compiler and visualizer application.

## Project Structure

- `frontend/`: Contains the frontend files (HTML, CSS, JavaScript)
  - `index.html`: Main HTML page
  - `styles.css`: Stylesheet
  - `app.js`: Frontend JavaScript logic
- `backend/`: Contains the backend files (Node.js)
  - `server.js`: Main server file
  - `compiler/lexRunner.js`: Module for running Lex compiler

## Features

- Interactive Lex Visualizer: Explains lexical analysis internally
- Online Lex Compiler: Write, compile, and run Lex programs in the browser

## Getting Started

1. Navigate to the `backend/` directory and run `npm install` to install dependencies.
2. Start the server with `node server.js`.
3. Open `frontend/index.html` in a browser to access the application.
